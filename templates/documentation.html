Grepolympia Calculator Documentation

What is this app?

This app helps you find the best way to distribute your athlete's points in Summer Grepolympia events for optimal results. It uses real event data and machine learning to suggest the best attribute allocation for maximum score.

How to use the app

  1. Select event: Choose the Summer Grepolympia event you want to optimize for from the dropdown menu.
  2. Enter athlete level: Input the number of points you have available to distribute among the attributes.
  3. Click "Calculate Optimal Distribution": The app will show you the best way to allocate your points for the highest estimated score.

What do the fields mean?

  • Select event: The specific Summer Grepolympia event (e.g., Hoplite Race, Archery).
  • Athlete level: The total number of points you can distribute among the three attributes.

Understanding the results

  • Best Distribution: The recommended number of points to assign to each attribute.
  • Estimated score: The score you are likely to achieve with this distribution. (This needs a bit more data to be accurate)

How does the prediction algorithm work?

The app uses a smart model selection process to give you the best possible predictions:

  • The event data is split into 70% for training, 20% for validation, and 10% for testing.
  • Two models are trained: a Random Forest and a Polynomial Regression (degree 2).
  • The model that performs best on the validation set is automatically chosen for predictions.
  • For athlete levels within the data range, predictions are highly accurate (the app has seen these cases before).
  • If you enter a level higher than the available data, the app will show a warning and do its best to extrapolate, but results may be less reliable.
This approach ensures you get the most accurate and robust predictions possible, using both machine learning and careful model selection. If you have more event data that could improve the model, please contact me.
